## 안녕하세요

숭실대학교 컴퓨터학부에서 공부하며 백엔드, 플랫폼, AI 시스템을 만듭니다. 서버 코드에
머무르지 않고 데이터 수집과 API부터 배포, 관측, 실패 복구까지 이어지는 흐름을 설계합니다.

기술의 수보다 문제와 제약, 선택한 이유와 검증 결과를 설명할 수 있는 엔지니어링을 지향합니다.
현재는 신입·주니어 백엔드, 플랫폼, AI 백엔드 기회를 찾고 있습니다.
